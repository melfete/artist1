# 🎵 Spotify Higher Lower Game

Ein interaktives Web-Spiel basierend auf Spotify-Streaming-Daten. Teste dein Wissen darüber, welcher Künstler mehr monatliche Hörer hat! Das Projekt ist für die Ausführung auf PHP-basierten Webservern optimiert.

## 🎮 Features

* **Echtzeit-Vergleich:** Vergleiche zwei zufällig gewählte Künstler aus der Datenbank.
* **Score-System:** Sammle Punkte für jede richtige Antwort und versuche, deinen Rekord zu brechen.
* **Spotify Design:** Modernes Dark-Mode Interface im typischen Spotify-Stil.
* **Responsive Layout:** Das Spiel funktioniert auf Smartphones, Tablets und Desktop-PCs.
* **Künstler-Mediathek:** Eine Übersicht aller im Spiel enthaltenen Künstler.

## 🛠️ Technische Details

* **Sprachen:** PHP 8.x, HTML5, CSS3.
* **Datenbank:** MySQL zur Speicherung der Künstler und Streams.
* **Hosting:** Veröffentlicht via InfinityFree Webhosting.
* **Bild-Logik:** Automatisches Finden von Künstlerbildern durch bereinigte Namenskonventionen.
